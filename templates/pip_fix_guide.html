إصلاح مشكلة "No module named pip"
المشكلة: عند محاولة تثبيت مكتبات مثل transformers، قد تظهر رسالة الخطأ التالية: 
Failed to install transformers: c:\Users\Eg2\Documents\trae pro\1\ai-translator1\venv\Scripts\python.exe: No module named pip
الحل السريع

تم توفير سكريبتات لإصلاح هذه المشكلة تلقائيًا:

لمستخدمي Windows
  1. أغلق تطبيق المترجم الآلي إذا كان قيد التشغيل
  2. انقر نقرًا مزدوجًا على ملف fix_pip.bat في مجلد التطبيق
  3. انتظر حتى اكتمال العملية
  4. أعد تشغيل تطبيق المترجم الآلي
تنزيل fix_pip.bat
لمستخدمي Linux/Mac
  1. أغلق تطبيق المترجم الآلي إذا كان قيد التشغيل
  2. افتح Terminal
  3. انتقل إلى مجلد المترجم الآلي
  4. قم بتنفيذ السكريبت: bash fix_pip.sh
  5. انتظر حتى اكتمال العملية
  6. أعد تشغيل تطبيق المترجم الآلي
تنزيل fix_pip.sh
الحل اليدوي

إذا لم تعمل السكريبتات، يمكنك حل المشكلة يدويًا باتباع الخطوات التالية:

لنظام Windows
  1. افتح موجه الأوامر (Command Prompt)
  2. انتقل إلى مجلد المترجم الآلي
  3. قم بتنشيط البيئة الافتراضية: 
    venv\Scripts\activate
  4. قم بتثبيت pip: 
    python -m ensurepip --upgrade
    أو 
    python -m ensurepip --default-pip
  5. قم بترقية pip: 
    python -m pip install --upgrade pip
  6. قم بتثبيت المكتبات المطلوبة: 
    python -m pip install transformers torch torchaudio
لنظام Linux/Mac
  1. افتح Terminal
  2. انتقل إلى مجلد المترجم الآلي
  3. قم بتنشيط البيئة الافتراضية: 
    source venv/bin/activate
  4. قم بتثبيت pip: 
    python -m ensurepip --upgrade
    أو 
    python -m ensurepip --default-pip
  5. قم بترقية pip: 
    python -m pip install --upgrade pip
  6. قم بتثبيت المكتبات المطلوبة: 
    python -m pip install transformers torch torchaudio
ملاحظات إضافية

  • تأكد من أن لديك اتصال بالإنترنت أثناء تنفيذ هذه الخطوات
  • قد تحتاج إلى صلاحيات المسؤول (Administrator) لتنفيذ بعض هذه الأوامر
